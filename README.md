 generacion-8bit 🕹️

Repositorio de juegos y proyectos interactivos hechos con HTML5, canvas y JavaScript puro.

Este es mi rincón de desarrollo de juegos — sin engines, sin frameworks, solo código desde cero.

**Stick vs Ticks** fue desarrollado durante mi primera semana en Linux, sin experiencia previa en desarrollo de juegos. Todo empezó por accidente y terminó siendo algo real.

---

**Tecnologías:** HTML5 Canvas · Web Audio API · JavaScript vanilla
